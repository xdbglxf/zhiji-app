# 知己 · 情绪疗愈花园

> 每天两分钟，被这个世界温柔以待。

**知己**是一款国风情绪疗愈 Web 应用，为需要情绪陪伴的年轻人打造。

🌐 **在线体验**：[https://xdbglxf.github.io/zhiji-app](https://xdbglxf.github.io/zhiji-app)

---

## ✨ 功能特色

| 功能 | 说明 |
|------|------|
| 🌸 **今日夸奖** | 每天一句专属鼓励，国风美文治愈你 |
| 🎁 **惊喜盲盒** | 来自陌生人的温暖心意，拆开就是暖意 |
| 💭 **心情打卡** | 记录每日情绪，即时给出暖心反馈 |
| 📅 **心情日历** | 可视化情绪轨迹，看见自己的成长 |
| 🌱 **成长花园** | 每日浇水，见证心情开出花来 |
| ✉️ **时光信箱** | 写信给未来的自己，等风来，也等你 |
| 🧘 **呼吸练习** | 30秒正念呼吸，让焦虑安静下来 |
| 🎵 **古风音乐** | Web Audio 合成五声音阶，沉浸式治愈 |

---

## 🎨 设计理念

- **国风美学**：水墨色彩 × 楷书字体 × 宣纸质感
- **情绪价值优先**：打开即治愈，2秒感受到温暖
- **零负担交互**：不强制打卡，不催促连续，随时来随时走

---

## 🚀 快速开始

纯静态 HTML，无需安装任何依赖：

```bash
git clone https://github.com/xdbglxf/zhiji-app.git
cd zhiji-app
# 直接用浏览器打开 index.html 即可
```
